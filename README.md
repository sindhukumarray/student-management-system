# 🎓 Student Management System

## Overview
Student Management System is a DBMS Mini Project developed using Node.js, Express.js, MySQL, HTML, CSS, and JavaScript.

## Features

- Add Student
- View Students
- Update Student
- Delete Student
- Search Student
- Dashboard Statistics
- Responsive Design
- Form Validation

## Technologies Used

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MySQL

## CRUD Operations

- Create
- Read
- Update
- Delete

## Database

Database Name: studentdb

Table Name: students
